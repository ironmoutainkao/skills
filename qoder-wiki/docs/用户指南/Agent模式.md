# Agent 模式

来源: https://docs.qoder.com/zh/user-guide/chat/agent

## 概述

Agent 模式具备自主决策、环境感知和工具调度能力，适合端到端完成编码任务。

## 核心能力

- 项目级多文件改动
- 自动制定执行计划
- 自动识别项目框架、技术栈和错误信息
- 自主使用搜索、读写、终端等工具
- 支持自动发现和调用 MCP 工具

## 工作方式

### 规划

复杂任务会先生成方案供用户审阅，确认后再执行。

### 待办事项

Qoder 会生成待办列表并显示状态：

- 空心圆：未开始
- 旋转圆：进行中
- 复选框：已完成

### 执行命令

- 默认每条命令执行前需要确认
- 可在设置中配置自动执行允许列表
- 后台命令会显示运行状态并持续跟踪输出

### MCP 工具

- 已配置 MCP 后，Agent 可按需调用
- 执行前通常会征求确认

## 额外功能

- 一键优化提示词
- 多轮会话持续迭代
- 与 diff 审阅、快照回滚结合使用
