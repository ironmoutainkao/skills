# 聊天概览

来源: https://docs.qoder.com/zh/user-guide/chat/overview

## 核心能力

Qoder 的智能会话包含三种模式：

- 智能问答
- 智能体
- 专家团（Experts）

整体能力包括：

- 多种聊天模式自由切换
- 自动环境感知
- 工具调用
- 命令执行
- 多文件项目级修改
- 长期记忆

## 开始新聊天

- 打开会话面板：`Cmd/Ctrl + L`
- 在面板中选择聊天模式
- 输入结构化需求，并附带必要上下文

## 输入建议

- 明确目标和预期结果
- 补充文件、图片、代码变更等上下文
- 说明技术栈、规范和输出要求
- 通过多轮反馈迭代需求

## 代码修改与审阅

在智能体模式下，Qoder 支持：

- 多文件生成与应用
- 查看 diff
- 逐处接受或拒绝修改
- 在多轮会话中继续追加需求
- 使用快照回滚

## 相关能力

- 上下文：文件、目录、图片、提交、规则等
- 记忆：自动沉淀个人和项目偏好
- 工具：搜索、读写、命令、错误排查
- MCP：接入外部工具与服务
