# 工具

来源: https://docs.qoder.com/zh/user-guide/chat/tools

## 概述

Qoder 内置多类工具，覆盖搜索、编辑、终端、问题定位与记忆更新，并可接入 MCP 工具。

## 工具分类

### 搜索

- 代码库搜索
- 文件搜索
- 代码片段搜索
- 目录浏览
- Web 搜索
- 网页内容抓取

### 编辑

- 修改文件
- 查看文件

### 执行命令

- 运行命令
- 获取终端输出

### 查找问题

- 获取项目中的代码问题

### 更新记忆

- 根据用户输入更新持久化记忆

### MCP 工具

- 按描述自主选择 MCP 工具
- 执行前通常请求确认
- 可结合自动运行配置减少人工干预
