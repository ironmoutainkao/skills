# 自定义模型

来源: https://docs.qoder.com/zh/user-guide/chat/custom-models

## 概述

Qoder 支持通过 API Key 接入第三方模型资源。

- 适用版本：`Pro Trial / Pro / Pro+ / Ultra`
- 当前文档列出的服务商：阿里云百炼、智谱、Kimi、MiniMax
- Repo Wiki 使用固定模型，仍会独立消耗 Qoder Credits

## 添加步骤

1. 打开 `Qoder 设置`
2. 进入左侧 `模型` 面板
3. 点击 `+ 添加模型`
4. 选择服务商并填写 API Key
5. 验证连接后开始使用

## 管理能力

- 编辑模型别名
- 删除模型
- 启用或禁用模型显示

## 常见问题

- 自定义模型本身不消耗 Qoder Credits，费用由模型服务商结算
- 支持为同一服务商添加多个模型或多个密钥
- 添加失败时重点检查 API Key、余额、权限和网络
