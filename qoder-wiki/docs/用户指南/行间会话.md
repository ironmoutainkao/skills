# 行间会话

来源: https://docs.qoder.com/zh/user-guide/inline-chat

## 概述

行间会话（Inline Chat）允许在编辑器内直接对当前文件发起对话，适合就地修改代码。

## 打开方式

- macOS：`Cmd + I`
- Windows：`Ctrl + I`

## 上下文

在行间会话里输入 `#` 可添加上下文，例如：

- 文件
- 规则

## 典型用法

- 选中代码片段，要求重构、补注释或修复
- 不选中代码，直接要求生成当前文件中的新片段

## 适合场景

- 单文件局部修改
- 快速代码改写
- 小范围补全与增强
